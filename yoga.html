LENOVO YOGA 11S (2013)

 

 

 

The old version of yoga 11 uses NVIDIA TEGRA 3 processor and run's on windows RT. The upcoming ultrabook yoga 11s might be available till june 2013. It uses intel i processor and run's on windows 8.

processor - intel i processor (upto i7)

Ram - 8 GB

Hard Drive - 256 GB SSD.

Graphics - Intel HD 4000.

weight - 3.08 pounds.

Thickness - 0.67 inches

Operating System - windows 8.

Screen Size - 11.6 inches.

Category - Ultrabook.

The build quality of the lenovo yoga 11s is great and the unique hinge mechanism that allows it to flip back make it a convertible tablet.
